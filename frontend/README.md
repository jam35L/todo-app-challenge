# Frontend (Angular 21)

The Angular workspace for the TODO app. **See the [root README](../README.md)** for how to
install, run, and test the whole project (frontend + backend together).

Quick reference — run from this directory:

| Command | What it does |
|---------|--------------|
| `npm install` | Install dependencies |
| `npm start` | Dev server at http://localhost:4200 |
| `npm test -- --watch=false` | Run the Vitest unit tests once |
| `npm run build` | Production build into `dist/` |

The backend API base URL is configured in `src/environments/environment.ts`
(defaults to `http://localhost:5080/api`).
